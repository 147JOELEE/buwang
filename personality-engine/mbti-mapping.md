## 1. 设计背景

### 为什么需要 MBTI 映射？

MBTI 是大众最熟悉的人格分类系统，在全球范围内拥有数亿用户。虽然学术界对 MBTI 的信效度存在争议，但它在以下场景中不可替代：

1. **用户认知**：普通用户更容易理解和选择 MBTI 类型
2. **入门引导**：作为新用户创建虚拟角色的快捷方式
3. **社区兼容**：方便与其他 MBTI 人格社区的内容互操作
4. **内容创作**：MBTI 标签在社交内容中广泛使用

### 设计目标

- 将离散的 MBTI 类型（16种）映射到连续的 Big Five 参数（0-100 x 5）
- 映射结果符合心理学文献中的相关性研究
- 允许用户微调映射结果（MBTI 作为起点，Big Five 作为精确表达）
- 映射算法透明、可复现、可调整

### 学术基础

映射设计参考了以下关键文献：
- McCrae, R.R., & Costa, P.T. (1989). *Reinterpreting the Myers-Briggs Type Indicator From the Perspective of the Five-Factor Model of Personality.* — 确立了 MBTI 四个维度与 Big Five 的对应关系
- Furnham, A. (1996). *The big five versus the big four.* — 量化了各维度间的相关系数
- MacDonald, D.A., et al. (1994). *The relationship between the Myers-Briggs Type Indicator and the NEO Personality Inventory.* — 提供了实证映射数据

## 2. 映射方法论

### 2.1 整体策略

采用**加权维度映射法 (Weighted Dimension Mapping)**：

```
MBTI四维度 (E/I, S/N, T/F, J/P) → 加权公式 → Big Five五维度 (O, C, E, A, N)
```

### 2.2 核心原理

1. **主导映射**：每个 MBTI 维度有一个主导对应的 Big Five 维度
2. **次要溢出**：每个 MBTI 维度同时对其他 Big Five 维度有次要影响
3. **加权求和**：将四个 MBTI 维度的贡献加权求和
4. **经验校准**：根据文献中的相关系数进行人工校准

### 2.3 映射流程

```
┌──────────┐    ┌──────────────┐    ┌──────────────┐    ┌──────────────┐
│  MBTI    │───▶│  加权公式    │───▶│  初步Big Five│───▶│  经验校准    │───▶  最终参数
│  类型    │    │  计算        │    │  参数值      │    │  (人工调整)  │
└──────────┘    └──────────────┘    └──────────────┘    └──────────────┘
```

## 3. 维度映射关系

### 3.1 E/I → 外向性 (Extraversion) — 主导映射

| MBTI | 方向 | 主要影响 | 次要影响 |
|------|------|----------|----------|
| E (外向) | +1 | 外向性 +17.5 (权重0.7×25) | 开放性 +3.75, 宜人性 +3.75 |
| I (内向) | -1 | 外向性 -17.5 (权重0.7×25) | 开放性 -3.75, 宜人性 -3.75 |

**理由**：E/I 与外向性的相关性最高 (r≈0.74)，但也与开放性的"社交开放性"和宜人性的"社交温暖"维度有关联。

### 3.2 S/N → 开放性 (Openness) — 主导映射

| MBTI | 方向 | 主要影响 | 次要影响 |
|------|------|----------|----------|
| N (直觉) | +1 | 开放性 +20.0 (权重0.8×25) | 尽责性 -5.0 (S型更尽责) |
| S (感觉) | -1 | 开放性 -20.0 (权重0.8×25) | 尽责性 +5.0 |

**理由**：S/N 与开放性的相关性在所有维度中最高 (r≈0.72)。S型倾向于务实和规则导向，因此与尽责性有正向关联。

### 3.3 T/F → 宜人性 (Agreeableness) — 主导映射

| MBTI | 方向 | 主要影响 | 次要影响 |
|------|------|----------|----------|
| F (情感) | +1 | 宜人性 +18.75 (权重0.75×25) | 神经质 +6.25 (F型情绪更敏感) |
| T (思考) | -1 | 宜人性 -18.75 (权重0.75×25) | 神经质 -6.25 |

**理由**：T/F 与宜人性的相关性 (r≈0.44)。F型更关注人际关系和谐，也与更高的情绪敏感性（神经质）相关。

### 3.4 J/P → 尽责性 (Conscientiousness) — 主导映射

| MBTI | 方向 | 主要影响 | 次要影响 |
|------|------|----------|----------|
| J (判断) | +1 | 尽责性 +17.5 (权重0.7×25) | 神经质 -3.75, 外向性 +3.75 |
| P (感知) | -1 | 尽责性 -17.5 (权重0.7×25) | 神经质 +3.75, 外向性 -3.75 |

**理由**：J/P 与尽责性的相关性 (r≈0.49)。P型因不确定性容忍度高而表现出更高的神经质，J型在社交中更主动。

## 4. 加权计算公式

### 4.1 通用公式

对于 Big Five 维度 d：

```
Score_d = 50 + Σ(mbti_dim_i.direction × weight_{i→d} × 25)
```

其中：
- `50` = 基准分（中点），保证所有类型在 0-100 范围内
- `mbti_dim_i.direction` = MBTI 维度方向（+1 或 -1）
- `weight_{i→d}` = 维度 i 对 Big Five 维度 d 的权重
- `25` = 每个 MBTI 维度的最大影响力（半个标准差范围）

### 4.2 各维度具体公式

**外向性 (E)**:
```
E = 50 + (E/I_direction × 0.70 × 25) + (J/P_direction × 0.15 × 25)
```

**开放性 (O)**:
```
O = 50 + (S/N_direction × 0.80 × 25) + (E/I_direction × 0.15 × 25)
```

**宜人性 (A)**:
```
A = 50 + (T/F_direction × 0.75 × 25) + (E/I_direction × 0.15 × 25)
```

**尽责性 (C)**:
```
C = 50 + (J/P_direction × 0.70 × 25) + (S/N_direction × 0.20 × 25)
```

**神经质 (N)**:
```
N = 50 + (T/F_direction × 0.25 × 25) + (J/P_direction × 0.15 × 25)
```

### 4.3 计算示例：INTJ

```
方向: I=-1, N=+1, T=-1, J=+1

E = 50 + (-1)×0.70×25 + (+1)×0.15×25 = 50 - 17.5 + 3.75 = 36.25
O = 50 + (+1)×0.80×25 + (-1)×0.15×25 = 50 + 20.0 - 3.75 = 66.25
A = 50 + (-1)×0.75×25 + (-1)×0.15×25 = 50 - 18.75 - 3.75 = 27.50
C = 50 + (+1)×0.70×25 + (-1)×0.20×25 = 50 + 17.5 - 5.0 = 62.50
N = 50 + (-1)×0.25×25 + (-1)×0.15×25 = 50 - 6.25 - 3.75 = 40.00

经验校准后:
E: 36.25 → 20 (I型下调，典型INTJ更为内向)
O: 66.25 → 75 (N型上调，直觉型更具开放性)
A: 27.50 → 30 (保持)
C: 62.50 → 80 (J型上调，INTJ以高度自律著称)
N: 40.00 → 40 (保持)
```

## 6. 经验校准规则

### 6.1 校准依据

基础加权公式提供了理论映射，但需要根据实证研究进行校准。校准规则基于以下原则：

1. **相关性强度优先**：与 Big Five 相关性更强的 MBTI 维度获得更大的校准幅度
2. **极端值合理性**：避免出现 0 或 100 的极端值（保留 5-95 范围）
3. **类型内部一致性**：同类型的各维度得分应符合该类型的整体人格画像

### 6.2 校准规则表

| 规则ID | 目标维度 | 触发条件 | 校准操作 | 幅度 | 依据 |
|--------|----------|----------|----------|------|------|
| CAL-001 | C (尽责性) | J型 | +10~15 | 上调 | r(J/P, C) ≈ 0.49 |
| CAL-002 | C (尽责性) | P型 | -5~10 | 下调 | r(J/P, C) ≈ 0.49 |
| CAL-003 | O (开放性) | N型 | +5~10 | 上调 | r(S/N, O) ≈ 0.72 |
| CAL-004 | O (开放性) | S型 | -5~10 | 下调 | r(S/N, O) ≈ 0.72 |
| CAL-005 | A (宜人性) | F型 | +10~15 | 上调 | r(T/F, A) ≈ 0.44 |
| CAL-006 | A (宜人性) | T型 | -5~10 | 下调 | r(T/F, A) ≈ 0.44 |
| CAL-007 | N (神经质) | I+N+F 组合 | +5~10 | 上调 | 文献中该组合显示更高神经质 |
| CAL-008 | E (外向性) | I型 | 限制≥5 | 下限保护 | 防止完全静默 |
| CAL-009 | E (外向性) | E型 | 限制≤95 | 上限保护 | 防止过度活跃 |

### 6.3 校准优先级

当多个校准规则冲突时，按以下优先级：
1. 上限/下限保护（CAL-008, CAL-009）— 最高优先级
2. 相关性强度校准（CAL-001~006）— 按相关性系数排序
3. 类型组合校准（CAL-007）— 最低优先级

## 7. 使用场景

### 7.1 场景一：用户通过 MBTI 创建角色

```
用户选择 "INFJ" → 查找映射表 → 获得 Big Five: {O:78, C:75, E:25, A:78, N:55}
→ 用户可在此基础上微调每个维度 ±10
→ 生成角色的 System Prompt
```

### 7.2 场景二：角色 MBTI 标签展示

```
角色 Big Five: {O:85, C:80, E:40, A:65, N:20}
→ 计算与16种MBTI类型的欧氏距离
→ 最近类型: INTJ (距离 = 0.15)
→ 显示标签: "INTJ型 (建筑师)"
→ 如果距离 > 0.3: 显示为 "混合型 - 偏向 INTJ"
```

### 7.3 场景三：MBTI 社区内容导入

```
外部MBTI角色描述 "ENFP型元气少女"
→ 映射为 Big Five: {O:88, C:30, E:88, A:78, N:48}
→ 根据描述中的"元气少女"标签额外调整: E+5, N-5
→ 最终参数: {O:88, C:30, E:93, A:78, N:43}
→ 生成 System Prompt
```

### 7.4 场景四：反向推断 MBTI

```
从聊天文本中提取 Big Five: {O:58, C:28, E:22, A:72, N:48}
→ 计算与16种MBTI的相似度
→ 最匹配: ISFP (距离 = 0.05)
→ 次匹配: INFP (距离 = 0.18)
→ 显示: "ISFP型 (探险家)，置信度: 高"
```

## 8. 设计决策记录

### D-101: 使用 0-100 而非标准 T 分数

**决策**：使用 0-100 直观分数而非心理学标准 T 分数 (M=50, SD=10)
**理由**：
- 0-100 对用户和开发者更直观
- 避免"正态分布"假设（虚拟人格不一定服从正态分布）
- 方便与滑块UI直接对应
- 内部可转换，不影响专业性

### D-102: 保留经验校准步骤

**决策**：在加权公式后增加人工校准步骤
**理由**：
- 纯数学公式无法完全捕捉 MBTI 类型的细微特征
- 心理学文献中的相关系数存在研究间差异
- 某些类型（如 INTJ 的尽责性）被广泛认为高于公式预测
- 校准规则文档化，保证透明和可复现

### D-103: 允许用户微调映射结果

**决策**：MBTI 映射作为起点，允许用户 ±15 范围内微调
**理由**：
- MBTI 是粗略分类，个体差异需要微调空间
- 同一 MBTI 类型的两个人可能有不同的行为表现
- 微调范围限定 ±15，防止脱离 MBTI 基本框架

### D-104: 映射表的单向优先性

**决策**：MBTI → Big Five 是主映射方向，Big Five → MBTI 仅作标签参考
**理由**：
- Big Five 是更精确的模型，不应被 MBTI 限制
- 反向推断仅供展示用途，不参与人格引擎计算
- 保持 Big Five 作为唯一权威参数源
